    About & Privacy

    • API keys are stored only in this browser's local extension storage — never synced, never sent to any server other than the AI provider you selected.
    • Screenshots are sent directly from your browser to the AI provider you configured, using their official API.
    • All AI requests are made from the extension's background service worker, not from web pages, so page scripts can never read your API key.
    • History is stored locally and can be cleared at any time from the History tab.
